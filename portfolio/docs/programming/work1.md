# Решение задачи Two Sum

## Цель работы

Разработать программу на языке Python для поиска двух элементов массива, сумма которых равна заданному числу.

---

## Постановка задачи

Дан массив целых чисел `nums` и целое число `target`.
Необходимо вернуть индексы двух чисел, сумма которых равна `target`.

### Условия задачи

* существует не более одного решения;
* один и тот же элемент нельзя использовать дважды;
* если решения нет — вернуть пустой список.

### Примеры

#### Пример 1

```text
Ввод: nums = [2, 7, 11, 15], target = 9
Вывод: [0, 1]
```

#### Пример 2

```text
Ввод: nums = [3, 2, 4], target = 6
Вывод: [1, 2]
```

#### Пример 3

```text
Ввод: nums = [3, 3], target = 6
Вывод: [0, 1]
```

---

## Описание алгоритма

Для решения задачи используется метод на основе хэш-таблицы (словаря).

### Сложность

| Характеристика | Значение |
|---|---|
| Временная сложность | O(n) |
| Пространственная сложность | O(n) |

По сравнению с методом полного перебора (O(n²)) данный алгоритм существенно эффективнее на больших массивах.

### Алгоритм работы

1. Создать пустой словарь `num_map` для хранения уже просмотренных элементов.
2. Перебрать элементы массива с их индексами.
3. Для каждого элемента вычислить `complement = target - num`.
4. Если `complement` уже есть в словаре — найдена пара, вернуть `[num_map[complement], i]`.
5. Иначе — добавить текущий элемент в словарь: `num_map[num] = i`.
6. Если перебор закончен и пара не найдена — вернуть `[]`.

### Пример трассировки

Для `nums = [2, 7, 11, 15]`, `target = 9`:

| i | num | complement | num_map до | Действие |
|---|-----|------------|------------|----------|
| 0 | 2 | 7 | `{}` | добавляем 2 → `{2: 0}` |
| 1 | 7 | 2 | `{2: 0}` | 2 найден → возврат `[0, 1]` |

---

## Листинг программы

### Файл `two_sum.py`

```python
def two_sum(nums, target):
    num_map = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in num_map:
            return [num_map[complement], i]
        num_map[num] = i
    return []

# Пример использования программы:
print(two_sum([2, 7, 11, 15], 9))  # [0, 1]
print(two_sum([3, 2, 4], 6))       # [1, 2]
print(two_sum([3, 3], 6))          # [0, 1]
```
